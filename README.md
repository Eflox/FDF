## Compilation & Execution

To play the simulation
```
make re && ./fdf test_maps/mars.fdf
```

## Instructions

* ```WASD or ARROWS``` to move
* ```Q/W``` Increase/Reduce height multiplier
* ```-/=``` Zoom
* ```SPACE``` Stop rotation
* ```ESC``` To quit
